Mobile Menu Bar

An overview about Menu Bar page.

Menubar creates as a separate file in mobile output. It includes search, change font-size, change font-famiy and a back button. When user trigger a search,the results will append to <div id="searchResults">. It has an associated javascript file to handle selection menu options. The values will store with cookies. It uses jquery.cookies java script library for that purpose. When use select the option remember search word ,last search word will append when opening Menubar.html.
