Mobile Settings

An overview about Settings page.

Settings page also created as a separate file in mobile output. It includes some basic settings which can be change to possible predefined values. There is a associated javascript file with this html page. It is responsible for set default values when initialising the mobile app. After that user can change the values also. It has the set of settings as following :

  • ToC : View the ToC

  • Menu bar : View the Menubar

  • Menu Bar Swipe Direction : Direction that trigger pop up Menu Bar (Up/Down)

  • ToC Swipe Direction : Direction that trigger pop up ToC (Up/Down)

  • Pop Up Menu Bar : Pop Up/not Menu Bar

  • Pop Up ToC : Pop Up/not ToC

  • Prev Page Direction : Swipe direction to move to Previous page

  • Next Page Direction : Swipe direction to move to Next page

  • Remember Last Search Word : Remember the last search word. If so, it will appear when move to menu bar

  • Reset : Reset the settings to default values
